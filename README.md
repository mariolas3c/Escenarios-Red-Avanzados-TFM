# Escenarios-Red-Avanzados-TFM
Recopilatorio de escenarios de red avanzados con Mininet

Para ejecutar los distintos escenarios es necesario disponer de una VM con [Mininet](https://www.google.com) instaldo.

En los distintos escenarios se utilizan topologias personalizadas definidas en python. Van acompañadas de un readme.md que explica la topologia y las pruebas a realizar para probar el entorno.
